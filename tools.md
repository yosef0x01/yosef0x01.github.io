## Open Source Tools

All MIT licensed. Use responsibly — only against systems you own or have authorization to test.

---

### grepsploit
Passive attack surface mapping at scale using only public data sources.

```bash
pip install grepsploit
grepsploit --target corp.com --output results.json
```

[GitHub →](https://github.com/yosef0x1/grepsploit)

---

### burp2obsidian
Burp Suite extension that auto-generates structured Markdown notes per endpoint.

[GitHub →](https://github.com/yosef0x1/burp2obsidian)

---

### fwdump
Firmware extraction and static analysis toolkit. Supports SquashFS, JFFS2, ext2/3/4.

```bash
python3 fwdump.py --image firmware.bin --extract --analyze
```

[GitHub →](https://github.com/yosef0x1/fwdump)

---

### tlsfuzz
Stateful TLS fuzzer for edge-case ClientHello parsing.

[GitHub →](https://github.com/yosef0x1/tlsfuzz)
